WIFI SETTINGS

By deafult, the BASIC ESP SERVER starts as an access point named espserver with the password 123456789. After the first connection you may change this setting using the form below. You can also hold down the reset button for some time to return to the access point mode. After setting the wifi behaviour, the curcuit is automatically restarted. You can always check out the mode leds to see how the wifi is being handled by the server.
